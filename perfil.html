Banered

Simple banner/icon maker.

Banered is a web app built with React and TypeScript. It allows users to easily create custom banners and icons. Key libraries used in this project include html2canvas for image generation, lucide-react for a rich set of icons, and react-color for intuitive color selection. The goal was to provide a user-friendly tool for quick visual asset creation.
